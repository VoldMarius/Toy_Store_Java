1)чтобы создать две новые директории в Linux - откройте терминал и используйте команду mkdir для создания директорий. Введите следующие команды:

mkdir "Игрушки для школьников"
mkdir "Игрушки для дошколят"

2)  Чтобы создать текстовые файлы в директории "Игрушки для школьников" - необходимо перейти в эту директорию и воспользоваться командой touch, например:

cd "Игрушки для школьников"
touch "Роботы" "Конструктор" "Настольные игры"

3) Для создания текстовых файлов в директории "Игрушки для дошколят" - нужно перейти в эту директорию и, аналогично, воспользоваться командой touch, например:

cd ../"Игрушки для дошколят"
touch "Мягкие игрушки" "Куклы" "Машинки"

4) Для объединения двух директорий в Linux - нужно использовать команду cp со свойством -r и указать исходную и целевую директории. Введите следующую команду в терминале:

cp -r "Игрушки для дошколят"/* "Игрушки для школьников"

5) Чтобы переименовать директорию в Linux - нужно использовать команду mv и указать старое и новое имя директории. Введите следующую команду в терминале:

mv "Игрушки для школьников"/ "Игрушки"

6) Для просмотра содержимого каталога "Игрушки" - используйте команду ls, например:

ls "Игрушки"

7) Чтобы установить и удалить snap-пакет в Linux, можно воспользоваться командами snap install и snap remove. Например, для установки пакета VLC введите команду:

sudo snap install vlc

Для удаления пакета VLC введите команду:

sudo snap remove vlc

8)добавим произвольную задачу для выполнения каждые 3 минуты. Для этого нужно создать скрипт, который будет выполнять задачу, и настроить его запуск при помощи утилиты cron.
Создадим скрипт с помощью команды nano:

nano myscript.sh

Вставим в него следующий код:

#!/bin/bash
date > ~/Documents/mytext.txt

После этого сохраним и закроем файл.

Затем нужно добавить задачу в cron. Для этого введите команду:

crontab -e

Откроется файл cron, в котором нужно добавить строку:

*/3 * * * * bash ~/myscript.sh

Эта строка запускает скрипт каждые 3 минуты. Сохраните и закройте файл.

Теперь каждые 3 минуты скрипт будет записывать текущую дату и время в файл mytext.txt в директории ~/Documents.
