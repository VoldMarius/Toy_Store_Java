1)чтобы создать две новые директории в Linux - откройте терминал и используйте команду mkdir для создания директорий.

mkdir "Игрушки для школьников"
mkdir "Игрушки для дошколят"

2)  Чтобы создать текстовые файлы в директории "Игрушки для школьников" - необходимо перейти в эту директорию
 и воспользоваться командой touch

cd "Игрушки для школьников"
touch "Роботы" "Конструктор" "Настольные игры"

3) Для создания текстовых файлов в директории "Игрушки для дошколят" - нужно перейти в эту директорию
воспользоваться командой touch

cd ../"Игрушки для дошколят"
touch "Мягкие игрушки" "Куклы" "Машинки"

4) Для объединения двух директорий в Linux - нужно использовать команду cp со свойством -r
и указать исходную и целевую директории.

cp -r "Игрушки для дошколят"/* "Игрушки для школьников"

5) Чтобы переименовать директорию в Linux - нужно использовать команду mv и указать старое и новое имя директории.

mv "Игрушки для школьников"/ "Игрушки"

6) Для просмотра содержимого каталога "Игрушки" -  команда ls:

ls "Игрушки"

7) Чтобы установить и удалить snap-пакет в Linux, можно воспользоваться командами snap install и snap remove.

sudo snap install vlc

Для удаления пакета VLC введите команду:

sudo snap remove vlc

8) скрипт, который будет выполнять задачу, запуск при помощи утилиты cron.
Создадим скрипт с помощью команды nano:

nano myscript.sh


#!/bin/bash
date > ~/Documents/mytext.txt


Добавим задачу в cron:

crontab -e

Откроется файл cron, в котором нужно добавить строку:

*/3 * * * * bash ~/myscript.sh

Эта строка запускает скрипт каждые 3 минуты.

Теперь каждые 3 минуты скрипт будет записывать текущую дату и время в файл mytext.txt в директории ~/Documents.
